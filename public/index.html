Importing sequence files

The primary advantage to using Plasmid is the ability to access your sequences any time. Using Dropbox, you can easily import and export files, keeping your work consistent between your iPhone and pc.

Plasmid uses Genbank formatted sequence files. We like Genbank because it provides a high level of detail about a sequence, and the files can be read and understood easily by human readers. An example file can be found here.

  1. Open the app on you iDevice
  2. Navigate to the file page by selecting the folder icon
  3. Select "Link Dropbox" and enter your Dropbox account information
    • A folder "Apps/Plasmid" will automatically be created in your Dropbox
  4. Drag files into this folder to add them to Plasmid
    • Sequence files can be imported in either Genbank or Fasta format format, but Genbank format is prefered
      • Genbank files must use the extension "gb"
      • Fasta files must use the extension "fasta"
  5. Select files from the files menu to view them in the app

Adding and modifying annotations

Adding an annotation

  1. Using two fingers, select a section of the sequence that you would like to annotate
  2. Selected the "ooo" button
  3. In the menu that appears, select "Annotate"
    • The annotate page will open. Here you can set a new feature's Genbank key, label, sequence, and color
  4. Select the "Key" button, and choose a feature key on the page that opens
    • You can search through available keys using the search bar
    • Key descriptions will appear in the text area
    • Keep in mind that every feature must
  5. Click "Done" when you are finished
  6. Set the feature color and label
  7. If you'd like, you can modify the sequence itself by selecting the sequence text area
  8. Click the "+" button to add your feature to the sequence
    • The new feature will only be added if it is found in your sequence
    • If multiple copies of the feature's sequence are found, they will all be marker be marked and annotated as one feature
      • In the Genbank file, multiple occurances will be condenced into one feature annotation, with the positions of seperate occurrances marked by seperate position attributes joined by the "order" keyword
      • This is done to save space and facilitate rapid manipulation of features

Modifying an annotation

  1. Select the "Features" button
  2. Adjust feature colors and modify labels
    • Changes will be saved automatically

Feature scanning

One of the most powerful features of Plasmid is the ability to scan your sequence for predefined features. Default feature lists are package with the app, but you can load your own custom lists easily.

Scanning for default features

  1. Select the "ooo" button
  2. In the menu that appears, select "Scan"
  3. Select the default libraries you would like to use from the library menu
  4. Select "Scan selected" to scan using selected libraries or choose "Scan all" to scan using all available libraries
    • Depending on the number of libraries available, using "Scan all" may take some time

Adding custom feature lists

You can easily create your own feature lists to use for scanning new sequences. Sharing feature lists with lab members is a great way to standardize your annotation schemes. An example feature list file can be found here.

  1. Using any spreadsheet editor, create a table with 3 columns
  2. Each feature will be defined in one row
    • Column 1 will contain the feature's label or name
    • Column 2 will contain the feature's key
      • This must be a valid Genbank feature key
    • Column 3 will contain the feature's sequence
      • The sequence can contain only the letters "A," "T," "G," and "C"
  3. Save your spreadsheet with the extension "csv"
  4. Drag the "csv" file into the Plasmid folder in your Dropbox

Inspecting sequences and mapping features

Plasmid offers the ability to quickly inspect regions of a sequence for paramaters such as melting temperature, length, and encompassed features. In addition, Plasmid can generate graphical maps of your sequence that allow for rapid visualization of features' relative positions.

Inspecting sequences

  1. Select a region of the sequence using two fingers
  2. Select the "ooo" button
  3. Select "Inspect" or "Translate" to view information about or a translation of the selected sequence

Mapping features

  1. Select "Map" to view a circular map of your sequence
  2. Select features from the table to highlight them on the circular map
